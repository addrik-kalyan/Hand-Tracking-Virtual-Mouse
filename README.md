# Hand Tracking Virtual Mouse

A computer vision project that controls the mouse cursor using hand gestures using OpenCV, MediaPipe, and PyAutoGUI.

## Features

- Real-time hand tracking
- Cursor movement
- Mouse click
- Drag and drop
- Scroll up/down
- Pause gesture

## Technologies

- Python
- OpenCV
- MediaPipe
- PyAutoGUI

## Run

pip install -r requirements.txt

python virtual_mouse.py
